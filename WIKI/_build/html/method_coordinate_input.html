Model input for s-coordinate¶

  • Double-power s-coordinate
\[s = C_1 \sigma^{A_1} + C_2 \sigma^{A_2}\]

where \(C_1\) and \(A_1\) make finer upper layers, and \(C_2\) and \(A_2\) make finer lower layers. In input.txt, we use ‘upper’ and ‘lower’ represent ‘1’ and ‘2’, respectively.

EXPONENTIAL = T
C_upper = 1.0
C_lower = 0.0
A_upper = 0.7
A_lower = 0.0
  • Hyperbolic s-coordinate
\[s = \tanh(\alpha \sigma) / \tanh (\alpha)\]

where \(\alpha\) is a parameter.

HYPERBOLIC = T
HYPER_alpha = 1.5
