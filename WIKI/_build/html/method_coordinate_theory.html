New s-Coordinate¶

Let \(s(x'_\alpha,z',t')\) be the replacement for \(z'\) in a new coordinate system \((x,y,s,t)\), with a range \(0 \leq s \leq 1\) that preserves the boundary locations from the \(\sigma\) coordinate mapping. Following the same procedure as used in the \(\sigma\) transformation leads to the modified system of equations

\[\begin{split} D_{,t} + ( D u_\alpha )_{,\alpha} + \tilde{\omega}_{,s} &= \Gamma_s \\ (D u_\alpha)_{,t} + (D u_\alpha u_\beta + \frac{1}{2} gD^2 \delta_{\alpha\beta} )_{,\beta} + (u_\alpha \tilde{\omega})_{,s} &= S_{q\alpha} + S_{h\alpha} + u_\alpha \Gamma_s \\ (Dw)_{,t} + (D u_\beta w)_{,\beta} + (w\tilde{\omega})_{,s} &= S_{qz} + w \Gamma_s\end{split}\]

in coordinates \((x,y,s, t)\). The modified normal velocity \(\tilde{\omega}\) is given by

(1)¶\[ \tilde{\omega} = D ( s_{,t} + u_\alpha s_{,\alpha} + w s_{,z} )\]

and the inhomogeneous term \(\Gamma_s\) is given by

(2)¶\[\Gamma_s = D_{,t} + u_\alpha D_{,\alpha} + D ( (s_{,t})_{,s} + u_\alpha (s_{,\alpha})_{,s} + w (s_{,z})_{,s} )\]

The term \(\Gamma_s\) complicates the set of equations unless we are able to find transformations which reduce \(\Gamma_s\) to zero, as occurs for the original choice \(s = \sigma\). Such a choice does not appear to be possible. A convenient choice which minimizes the impact on the structure of the governing equations is to let \(s\) be specified entirely as a function of \(\sigma\), considered further in the cases later. For \(s(\sigma)\) only, we obtain the derivatives

\[s_{,t} = s_{,\sigma} \sigma_{,t}; s_{,\alpha} = s_{,\sigma} \sigma_{,\alpha}; s_{,z} = s_{\sigma} \sigma_{,z}\]

In this case, the normal velocity is given by \(\tilde{\omega} = s_{,\sigma} \omega\) with \(\omega\) given by

\[\omega = D ( \sigma_{,t} + u_\alpha \sigma_{,\alpha} + w \sigma_{,z} ),\]

and thus \(\tilde{\omega}\) satisfies the same boundary conditions as \(\omega\) at the upper and lower boundaries. Using the substitution for the derivatives of \(s\) in (2) allows \(\Gamma_s\) to be written as

\[\Gamma_s = \Gamma_\sigma - (\sigma_{,s})_{,\sigma}\tilde{\omega} \label{eq19}\]

where \(\Gamma_\sigma = 0\) as stated below. Further, the combination of the \(s\) derivative term and the residual reduces to

\[\tilde{\omega}_{,s} - \Gamma_s = s_{,\sigma} ( \sigma_{,s} \tilde{\omega} )_{,s}\]

which is equivalent to the term \(\omega_{,\sigma}\) in the original equation. The full set of equations, retaining \(s\) derivatives and expressions in terms of \(s\) rather than \(\sigma\), are given by

\[D_{,t} + ( D u_\alpha )_{,\alpha} + s_{,\sigma}(\sigma_{,s}\tilde{\omega})_{,s} = 0\]
\[(D u_\alpha)_{,t} + (D u_\alpha u_\beta + \frac{1}{2} gD^2 \delta_{\alpha\beta} )_{,\beta} + s_{,\sigma}(\sigma_{,s} u_\alpha \tilde{\omega})_{,s} = S_{q\alpha} + S_{h\alpha}\]
\[(Dw)_{,t} + (D u_\beta w)_{,\beta} + s_{,\sigma}(\sigma_{,s} w \tilde{\omega})_{,s} = S_{qz}\]

Use of the transformation and retention of \(s\) rather than \(\sigma\) in all expressions leads at most to the introduction of a variable coefficient outside the \(s\)-derivative terms in each equation. For cases where \(s\) or \(\sigma\) can’t be expressed in terms of each other alone, the details of the transformation and the resulting expressions for \(\tilde{\omega}\) and \(\Gamma_s\) must be worked out for each individual case, as shown in the examples in the Section next.
